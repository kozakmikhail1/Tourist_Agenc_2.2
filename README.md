# Туристическое агентство

Программа для управления туристическим агентством, разработанная на C++ и Qt.

# Описание задания 
Программа должна иметь удобный пользовательский интерфейс с необходимыми пунктами меню. Информация должна храниться в нескольких файлах, связанных определенным образом. Программа должна обеспечивать работу туристического агентства по организации путешествий и отдыха людей в различных странах. Программа должна содержать:
- список стран, в которых оказываются туристические услуги;
- список компаний, осуществляющих перевозки людей в различные страны, а также график перевозок (дата отправления/прибытие);
- список отелей, с содержанием количества различных номеров и их стоимость;
Программа должна обеспечивать поиск необходимого варианта, подсчитывать стоимость поездки. После выбора варианта, удовлетворяющего клиента, осуществляется заказ на туристические услуги. Другие методы, в зависимости от специфики задачи. 
Разработать иерархию классов с использованием наследования. Разработать и использовать классы контейнеров, итераторов и алгоритмов(свои и STL). Производить обработку исключительных ситуаций.

## Структура проекта
Проект организован с разделением заголовочных (.h) и исходных (.cpp) файлов:

```
TouristAgenc_2.2/
├── include/              # Все заголовочные файлы (.h)
│   ├── models/          # Модели данных
│   ├── containers/     # Контейнеры и итераторы
│   ├── utils/          # Утилиты
│   ├── dialogs/        # Диалоги (GUI)
│   └── mainwindow/     # Главное окно
|       ├──  actions/   # Операции
├── src/                 # Все исходные файлы (.cpp)
│   ├── models/         # Реализация моделей
│   ├── containers/     # Реализация контейнеров
│   ├── utils/          # Реализация утилит
│   ├── dialogs/        # Реализация диалогов
│   └── main.cpp        # Точка входа
└── CMakeLists.txt      # Файл сборки
```

## Реализованные темы курса

1. ✅ **Классы и объекты** - все классы в проекте
2. ✅ **Дружественные функции. Перегрузка операций** - операторы ввода/вывода, сравнения
3. ✅ **Наследование** - иерархия классов (3 уровня)
4. ✅ **Виртуальные функции. Абстрактные классы** - TouristService как абстрактный класс
5. ✅ **Шаблоны функций и классов** - DataContainer<T>, ContainerIterator<T>
6. ✅ **Генерация и обработка исключительных ситуаций** - FileException, StreamFileException
7. ✅ **Потоки ввода/вывода. Работа с файлами** - std::ifstream, std::ofstream, операторы <<, >>
8. ✅ **Классы контейнеры-итераторы. STL-контейнеры** - кастомные контейнеры и STL алгоритмы

## Возможности

- Управление списком стран
- Управление отелями с номерами различных типов
- Управление транспортными компаниями с графиком перевозок
- Создание и управление турами
- Поиск туров по различным критериям
- Расчет стоимости туров
- Создание заказов на туристические услуги со статусами обработки
- Сохранение и загрузка данных из файлов

## Требования

- Qt 6.x
- CMake 3.16 или выше
- Компилятор с поддержкой C++17

## Использование

1. Запустите программу
2. Используйте меню для добавления стран, отелей, транспортных компаний
3. Создавайте туры, выбирая страну, отель и транспорт
4. Используйте функцию поиска для поиска туров по критериям
5. Создавайте заказы на туры
6. Сохраняйте данные через меню "Файл" -> "Сохранить"

## Хранение данных

Данные сохраняются в текстовые файлы в директории `data/`:
- `countries.txt` - страны
- `hotels.txt` - отели
- `transport_companies.txt` - транспортные компании
- `tours.txt` - туры
- `orders.txt` - заказы
